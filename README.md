# Project Overview

This project is a monorepo for a full-stack application using React for the frontend and Express for the backend.

## Setup Instructions

1. Clone the repository:
   
   ```bash
   git clone https://github.com/nardongputikyaneh/qjwcconstruction.git
   ```

2. Install dependencies for both frontend and backend:
   
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```

3. Start the servers:
   
   ```bash
   # Start frontend
   cd frontend && npm start

   # Start backend
   cd backend && npm start
   ```
